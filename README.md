## Execution
1. Install node.js
2. Install Vue.js and required dependencies
3. Clone repository (Select desired branch)
```
git clone https://github.com/xHugo21/imdb_academy.git
```
4. Run the project
```
cd imdb_academy
```
```
npm run dev
```
5. (Optional) Build the project
```
npm run build
```
